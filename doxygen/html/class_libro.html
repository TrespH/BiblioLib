Costruttore Libro.

Richiede 4 parametri in entrata: Titolo, Autore, Anno, Disponibilita.
In più se richiamato con valore booleano settato true (essendo parametro di default), provvede a generare
un nuovo Codice, calcolato mediante l'incremento di un contatore stampato su file binario, riscrivendolo aggiornato.
